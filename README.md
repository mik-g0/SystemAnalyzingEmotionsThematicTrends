# System Analyzing Emotions Thematic Trends

Веб-приложение для анализа пользовательских текстов с определением **эмоциональной окраски** и **тематической принадлежности** с использованием методов обработки естественного языка.

Система построена по клиент-серверной архитектуре: интерфейс на React взаимодействует с API на FastAPI, где выполняется анализ текста с применением предобученных NLP-моделей и сохраняются результаты в базе данных.

## Назначение проекта

Проект разработан в учебных целях для демонстрации практического применения методов искусственного интеллекта и анализа данных в веб-приложении.



## Возможности

- Регистрация и авторизация пользователей (JWT)
- Анализ текста на **эмоцию**
- Определение **темы текста**
- Сохранение истории анализов
- Просмотр истории через интерфейс
- REST API для взаимодействия фронтенда и бэкенда



## Используемые технологии

### Frontend
- React
- React Router
- Vite

### Backend
- FastAPI
- SQLite
- JWT-аутентификация

### NLP / ИИ
- Hugging Face Transformers
- Модель: j-hartmann/emotion-english-distilroberta-base
- Sentence-Transformers (all-MiniLM-L6-v2)
- Косинусное сходство для определения темы текста



## Архитектура работы

1. Пользователь вводит текст в интерфейсе.
2. Текст отправляется на API `/analysis`.
3. Модель определяет эмоцию текста.
4. Текст преобразуется в вектор (эмбеддинг) и сравнивается с векторами заранее заданных тематик.
5. Результат сохраняется в базу данных.
6. Пользователь может просмотреть историю запросов на странице History.

